# Benchmarkit LLM

This project was entirely vibe coded...

To get you started:

1 - Edit the questions.json file to add other questions

2 - Edit the endpoints.json file to point to the LLM's you have

3 - Initialize the project

```bash
python -m venv .venv
source .venv/bin/activate
pip install litellm
pip install streamlit
streamlit run benchmarkit.py
```
